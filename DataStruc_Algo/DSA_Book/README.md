# Data Structures & Algorithms — A Complete Study Book (C++ Edition)

> A dual-focus reference: **conceptual mastery** (language-agnostic) + **efficient modern C++ (C++14/17)** implementation.
> Built for deep learning, competitive programming, and technical interviews.

---

## How to Use This Book

- **Beginner?** Read in order: Foundations → Core Structures → Algorithms.
- **Interview prep?** Jump to [Problem-Solving Techniques](05_Problem_Solving.md) and [Interview Preparation](06_Interview_Preparation.md), then drill the [Cheatsheets](08_Cheatsheets.md).
- **Competitive programming?** Focus on [Advanced Data Structures](03_Advanced_Data_Structures.md) and [Algorithms](04_Algorithms.md).
- Every concept follows the same teaching loop: **WHY it exists → how it works internally → C++ code → tradeoffs → pitfalls.**

## Conventions Used

- Code targets **C++17** (compiles with `g++ -std=c++17 -O2`).
- `O(...)` is worst case unless stated; `Θ` tight bound; `Ω` lower bound.
- "STL equivalent" notes show the standard container/algorithm you'd use in practice.
- ⚠️ marks common pitfalls; 💡 marks key insights.
